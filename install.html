Installation on Ubuntu/Debian Linux


To find out your current Ubuntu version, use the following command:

lsb_release -a

Download the appropriate packages for your operating system from https://github.com/gisogrimm/tascar/releases.

To install TASCAR and all dependencies, navigate to the directory containing the downloaded packages and run:

sudo apt install ./path_to_packages/*.deb

Installation on macOS


TASCAR can be installed via homebrew. If not yet installed, then download and install 'homebrew' (https://brew.sh/). Then install all TASCAR with: 

brew install gisogrimm/tap/tascar

To start TASCAR, type 

tascar &
in the command window. Don't forget to start jack, e.g., with 
qjackctl &
before loading any session file.

Installation on Windows


A pre-compiled version is available for Windows. To install:

  1. Download the latest .zip archive from the GitHub releases page.
  2. Extract the downloaded archive to a folder of your choice.
  3. Install the JACK audio server using the installer provided on the JACK website.
  4. Start the JACK server (QjackCtl or similar).
  5. Run tascar.exe from the extracted folder to start TASCAR.

Starting TASCAR


To start TASCAR, first configure and start your jack server. Then, in a terminal, type 

tascar &
or start from "Sound and Video" applications menu.

If you intent to use TASCAR in scientific research projects, please remember to cite our open access publication:

Grimm, Giso; Luberadzka, Joanna; Hohmann, Volker. A Toolbox for Rendering Virtual Acoustic Environments in the Context of Audiology. Acta Acustica united with Acustica, Volume 105, Number 3, May/June 2019, pp. 566-578(13), doi:10.3813/AAA.919337

TASCAR is developed at the "Auditory Signal Processing and Hearing Devices" group at the Carl von Ossietzky Universität Oldenburg, Germany. The main author is Giso Grimm.
